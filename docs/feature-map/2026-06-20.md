## 四、下一阶段优先级

1. **真实模式集成**（最高）— 打通 `transport.on → events.dispatch` 到 useChat/useSidebar。mock 路径已验证，真实流式将 break 直到接通（W16 诊断发现）
2. **SubAgent 数据流**（Phase 3）— TaskTree / Side Drawer 接真实数据，`tool_execution_update` 事件接入
3. **聊天核心增强**（Phase 2）— @-mention、文件变更 Diff、Session Tree 可视化、系统通知
4. **树形引擎**（Phase 5，最高风险）— RPC 桥接 + ask_user + 递归分解 + 预算控制

长期 Phase 0–6 规划见旧版 [2026-05-19.md §三](2026-05-19.md)（规划层未变，仅完成度更新）。

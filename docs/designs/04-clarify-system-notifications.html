关键差异
A. 当前方案: done 和 info 用相同的绿色圆点+浅绿背景，alert 用红色圆点+浅红背景。仅 2 种视觉状态，warning/progress 缺失
B. 图标+边框: 每种类型有独立图标、独立背景色、独立边框色。4 种状态 (done/alert/warning/progress) 清晰区分
C. 内联状态行: 单行紧凑通知，带图标和颜色文字。适合轻量通知，告警操作用内联按钮。在消息流中不突兀

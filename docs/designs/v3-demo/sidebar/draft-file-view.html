§3文件树内过滤 · 实时（非 ⌘K）

File View 顶部过滤框：仅过滤当前树，命中高亮，非命中折叠。与 overlays/draft-search-modal 的全局 ⌘K 严格区分。

严格区分（handoff P0）：
本框 = 文件树过滤：实时、仅当前 session 改动文件、无命令/符号/会话。
⌘K = 全局 Overlay（overlays/draft-search-modal）：命令/文件/符号/会话四类，跨 session。
两者不混——本框无 ⌘K 提示，不弹浮层，输完即过滤。

§4文件操作 · 工具栏 + 右键

工具栏：新建文件 / 新建文件夹 / 刷新（ghost icon）。右键文件：重命名 / 删除 / 复制路径 / 在 Panel 打开。

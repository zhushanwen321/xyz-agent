完整链路
当用户选择 "max" 并发送消息时：
① UI InputToolbar 读取 model.thinkingLevelMap → 只显示非 null 的 level
② 用户选 max → rpc-client 发送 set_thinking_level: "max"
③ pi setThinkingLevel 内部 max → xhigh（pi-ai 只支持到 xhigh）
④ pi buildParams 读取 model.thinkingLevelMap.xhigh → "max" 作为 reasoning_effort
⑤ API 请求 → reasoning_effort: "max"

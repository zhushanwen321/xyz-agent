Data Flow
pi extension 调用
extension 在 TUI 中调用 setWidget(key, lines) 或 setStatus(key, text)
extension_ui_request { method: "setWidget", key: "goal-tracker", lines: ["✓ TC-1: ...", "○ TC-5: ..."] }
event-adapter.ts BRIDGE
拦截 extension_ui_request，根据 method 分发
case 'extension_ui_request': if (method === 'setWidget') → send WS event { type: 'extension.widget', payload: { widgetKey, lines } } if (method === 'setStatus') → callback + send WS event { type: 'extension.status', payload: { statusKey, text } }
WebSocket → event-bus
WS 消息到达前端，event-bus.ts 按 type 分发到 composable
useExtensionWidget.ts NEW
composable 消费事件，更新 reactive Map
// refCount protected on('extension.widget', onWidget) on('extension.status', onStatus) widgets = Map<key, payload> statuses = Map<key, payload>

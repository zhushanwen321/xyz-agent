§5实现要点

  • watermark：用 -webkit-text-stroke 描边 + color:transparent，低 opacity 0.045。比 SVG 轻、不影响 a11y（pointer-events:none）。
  • composer-area：contenteditable + min-height:40px + max-height:120px 自动滚动，placeholder 用 :empty:before 注入纯文字（文案与 panel composer 一致：描述你想让 AI 做什么，或 @ 引用、# 文件、/ 命令…）。
  • 工具条：左 c-insert（+ 添加）；右对齐 上下文容量 c-text.ctx / 模型 c-text.model / 思考 c-text.think / 发送 c-send——三者无边框、透明底、纯文字，hover 才浮 --bg-hover；发送是唯一实色。与 panel composer §1 共用同类名，不另造。
  • slash-token：与 panel/draft-composer-states.html §2e 同源，共用 .slash-token 类，不另造。
  • chip 选中态：用 .open 加 inset 0 0 0 1px var(--accent-ring)，与 sidebar / search modal 选中态同源，不画左色条（design-system §2 反模式）。
  • 窗口 data-fullscreen：监听 Electron enter/leave-full-screen 切换属性，traffic-light opacity 与 app-nav left 同步 320ms 平移（--duration-slow），参 shell/spec.md §二。
  • traffic light 安全区：.aside-region padding-top:52px，与 shell/spec.md §三 严格一致（mac 32 + 呼吸 20，三平台统一）。

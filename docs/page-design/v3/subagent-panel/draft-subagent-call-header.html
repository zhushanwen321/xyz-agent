字段说明（pi-subagent-workflow startParam schema，feat-ask-user-gui 分支）：

· task（必填）— 完整任务提示词，可多行。收起态取首行截断 60 字符预览，展开看全文

· slug（必填，≤20 字符）— 短标签，区分并发 subagent。如 fix-login-bug / extract-urls。与 task 是两个独立字段

· agent（可选，默认 general-purpose）— agent 类型

· model（可选）— 模型覆盖，省略继承主 agent

· thinkingLevel（可选）— off/minimal/low/medium/high/xhigh

· fork + worktree（可选）— 上下文继承 + 文件隔离，用彩色标签高亮

标题行：subagent {agent} · {slug} (model · thinking)，对齐 renderCall（tool-render.ts L101-119）。agent 和 slug 都是 accent 色。

实现前置：现有 xyz-agent 的 SubagentRecord（shared/src/subagent.ts）只有 task，缺 slug 字段。subagent-extractor.ts 需扩展，从 startParam.slug 提取并新增到 SubagentRecord。

会话项 · Session Item

Session List 内一行 = 状态点 + 标题 + 分支 pill + 目录·时间小字。L2 原子单元，复用于 sidebar Session List 与 overview 鸟瞰卡片。本叶只画单个会话项的全部状态与交互，不画整个 Sidebar 容器（容器见 draft-five-states.html）。

§1状态圆点 · 5 态

与 panel-header 状态圆点一致（design-system Status Dot：状态色填充，7px）。running / waiting 两态带脉冲环。命名以 sidebar/spec.md §会话项为准。

running 进行中
waiting 等输入/审批
done 完成
stopped 用户停
error 出错
裁决：handoff 列 4 态（running/idle/done/error），sidebar/spec.md §会话项列 5 态。以 spec.md 为准——补 waiting（带脉冲，等用户审批 steer）与 stopped（用户主动停，区别于 error 异常停）。颜色映射：进行中=accent 蓝、等待=warning 黄、完成=success 绿、停止=subtle 灰、出错=danger 红。

§2默认态 · 信息结构

状态点 + 标题（13px/550，超长省略）+ 分支 pill（mono + accent-soft）+ 时间（11px subtle）。单行基准高度 ~44px。

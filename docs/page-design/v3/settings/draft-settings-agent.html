设计说明 · 本页是 Agent 的唯一配置面：层 A 勾选加载路径（全局 + 项目，决策 global_configures_project），层 B 只读预览扫到的 subagent——无文件级开关、无 CRUD（目录在=启用，见 ADR-0020 §5）。要细粒度控制单个 agent，去目录里移除文件或用 pi 的命令。项目维度同样在此配（相对路径约定）。生效结果（哪些全局/项目 agent 实际加载、有无覆盖）在项目设置抽屉只读查看。
验收对照（handoff-agent.md §8 P0）：
① 层 A 强制目录灰显置顶、不可关不可拖 · ② 可选目录可勾选可拖排序，结果写 agentDirs · ③ 层 B 计数正确，多源 badge 链第一个标「生效」 · ④ 被覆盖来源展示路径 + 覆盖原因（强制>可选·项目>全局·靠前>靠后） · ⑤ 改 agentDirs 后提示「需重开会话生效」（ADR-0020 §5）

依赖方向（②§6 三层铁律）：transport→services（经 port）→infra（adapter 实现）；shared 被三层+前端共引。services 不 import infra IO（AC-1）；transport 不碰 node:内置（AC-2b，K-2 反哺扩展 grep）；前端 stores 间禁互引（K-9：跨 store 编排在 composables 层）。
